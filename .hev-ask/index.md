The Flox introduction: Flox is a next-generation, language-agnostic package and environment manager where you define everything an environment needs (packages, tools, environment variables, and services) in one manifest, switch between environments easily, share them, and keep them in version control. It uses the same setup across macOS and Linux on x86 and Arm, achieves isolation through pre-configured sub-shells rather than containers so it works with existing tools and dotfiles, and uses Nix under the hood for reproducibility without requiring you to learn Nix.
