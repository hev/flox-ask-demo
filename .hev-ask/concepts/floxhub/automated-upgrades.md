Describes automated upgrades as a paid Flox for Teams feature: organizations can let FloxHub upgrade environments automatically instead of upgrading manually. Explains how to set per-environment upgrade cadence (or disable it) and how to set an organization-wide upgrade policy, while cautioning against disabling automated upgrades for all environments due to outdated-dependency and security risks.
