Explains that once published, a package becomes available via search, show, and install, and is prefixed with the user or organization name to distinguish it from Base Catalog packages. When installed, the package is downloaded directly from the Catalog Store it was published to, and organizations using their own Catalog Store never have it downloaded to or cached on Flox servers.
