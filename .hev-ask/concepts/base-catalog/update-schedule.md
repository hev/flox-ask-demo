Describes the update cadence: Flox ingests the equivalent of nixos-unstable daily, so a package landing on nixos-unstable usually appears within a day. Distinguishes this from the separate, longer process between merging a nixpkgs pull request and a package reaching nixos-unstable, which can take days to over a week, and points to a tracker for following PR progress.
