Introduces package outputs, the named parts a catalog package is split into. Some packages have a single output while others have many, and which outputs exist and which are installed by default is set by the package maintainer upstream in Nixpkgs. The tutorial teaches discovering a package's available outputs, seeing which are default, and specifying exactly which to install.
