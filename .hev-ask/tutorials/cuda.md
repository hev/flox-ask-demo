Introduces using NVIDIA's CUDA Toolkit with Flox, a language-agnostic package and environment manager that gives reproducible subshell environments without container isolation. Flox is officially licensed to distribute the CUDA Toolkit, lets each project use a different CUDA version without conflicts, and eliminates the once-only effort of figuring out CUDA compatibility so anyone on Linux or WSL gets the same toolset.
