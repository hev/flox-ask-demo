Explains that this page applies only if an organization provides its own Catalog Store rather than the default one Flox provides. A user-provided Catalog Store is an S3 bucket or S3-compatible service, and configuring it requires setting ingress (upload) and egress (download) URIs for the catalog using a command-line utility provided by Flox.
