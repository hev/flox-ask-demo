Explains that with no arguments all packages are upgraded where possible (a package upgrades if its version, build configuration, or dependency graph changes), that packages can be upgraded by group name (with ungrouped packages in a default top-level group that can be upgraded alone), and that a single package can be upgraded by ID only if it doesn't share a group with others.
