Explains that the command creates a new empty environment (named after the directory by default), stores environment metadata in a local subdirectory, errors if one already exists, attempts to detect the project's language and suggest packages or activation scripts (auto-acceptable, and editable afterward), and when targeting the hub does not run directory-based setup hooks.
