Documents the build section of the manifest, which describes build instructions for custom packages built with Flox: a table of named build descriptors with a build command (run by Bash, with access to install packages and defined variables), a sandbox level (off, warn, enforce, or pure with their reproducibility and network/access tradeoffs), allowed-path patterns for the local sandbox modes, a version specified inline/from a file/from a command, a description, and a list of runtime packages to restrict runtime dependencies.
