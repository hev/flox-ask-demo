Documents the install table, the core of the manifest specifying which packages to install: the equivalent TOML spellings, the concept of the package path within the catalog, install IDs as the key for each package, and the catalog, flake, and store-path descriptor forms with their options (package group, version range, systems, package path, priority for resolving file conflicts, and output selection) and how each option constrains resolution.
