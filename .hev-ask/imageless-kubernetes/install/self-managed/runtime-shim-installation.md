Explains installing the runtime shim, recommending the automatic method (a Flox environment installer that detects the containerd version, installs the matching shim, updates containerd config, and restarts it) run as root on each node. Provides a manual method for unsupported distributions (such as those that vendor containerd in non-standard locations): create an environment, install the version-appropriate shim, symlink it, add the Flox runtime config to the containerd config matching its version, and restart containerd.
