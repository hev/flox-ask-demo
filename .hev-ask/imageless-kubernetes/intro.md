Introduces Imageless Kubernetes as a way to run Kubernetes backed by Flox environments rather than container images, replacing image pipelines and rebuilds with lightweight, reproducible environments centrally controlled via FloxHub. Because Flox environments provide the same dependencies everywhere, the tools used in development and CI match those running in a pod.
