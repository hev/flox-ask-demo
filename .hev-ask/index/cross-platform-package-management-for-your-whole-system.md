Describes using Flox for whole-system, cross-platform package management: set up a default environment of always-available packages across macOS and Linux on x86 or Arm, so setting up a new laptop or keeping machines in sync uses the exact same versions. Notes you can migrate from or use an existing package manager alongside Flox.
